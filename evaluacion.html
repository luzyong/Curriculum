Introduccion

¿Cómo funciona un contenedor?

Un contenedor es una sección perteneciente a ramas del arbol DOM. Cada sección puede tener un encabezado asociado y puede contener cualquier número de subsecciones anidadas adicionales. El algoritmo para el esquema también asocia cada nodo en el árbol DOM con una sección particular y potencialmente un encabezado. (Las secciones en el esquema no son elementos de sección, aunque algunos pueden corresponder a tales elementos, son simplemente secciones conceptuales).




Instalar Apache

Pasos a seguir

Si te diriges a la página de Apache encontrarás los siguientes pasos:

  • Descarga la última versión desde http://httpd.apache.org/download.cgi
  • Extraer $ gzip -d httpd-NN.tar.gz $ tar xvf httpd-NN.tar $ cd httpd-NN
  • Configura $ ./configure --prefix=PREFIX
  • llenar $ make
  • Instala $ make install
  • Personalizalo $ vi PREFIX/conf/httpd.conf
  • Prueba $ PREFIX/bin/apachectl -k start

HTML

¿Qué es un HTML?

HTML es el lenguaje de marcado básico de la World Wide Web. Originalmente, HTML fue diseñado principalmente como un lenguaje para describir semánticamente documentos científicos. Su diseño general, sin embargo, le ha permitido adaptarse, durante los años siguientes, para describir una serie de otros tipos de documentos e incluso aplicaciones.
Los documentos HTML se componen de un árbol de elementos y texto. Cada elemento se denota en la fuente por una etiqueta de inicio , tal como " body", y una etiqueta de fin , tal como " /body". (Ciertas etiquetas de inicio y finales pueden en ciertos casos ser omitidos y están implícitos en otras etiquetas).
Agentes de usuario HTML (por ejemplo, navegadores Web) analizan este marcado, convirtiéndolo en un árbol DOM (Document Object Model). Un árbol DOM es una representación en memoria de un documento.

Elemento

Los elementos pueden tener atributos, que controlan el funcionamiento de los elementos.
Las etiquetas contienen una etiqueta con su nombre , dando el nombre del elemento. Todos los elementos HTML tienen nombres que sólo utilizan caracteres ASCII alfanuméricos . En la sintaxis HTML, los nombres de etiqueta, incluso aquellos para los elementos extraños , pueden ser escritos con cualquier combinación de letras mayúsculas bajos y que, cuando se convierte a todos en minúsculas, coincide con el nombre de la etiqueta del elemento; nombres de las etiquetas son sensibles a mayúsculas.
Las etiquetas se utilizan para delimitar el inicio y el final de los elementos en el marcado. Texto sin formato , escapable texto sin formato , y los elementos normales tienen una etiqueta de inicio para indicar dónde empiezan y una etiqueta de cierre para indicar dónde terminan.
Los atributos de un elemento se expresan dentro de la etiqueta inicial del elemento.
Los atributos tienen un nombre y un valor. Los nombres de atributo deben constar de uno o más caracteres distintos de los caracteres de espacio, NULL, comillas ( "), apóstrofe ( '), signo mayor que (>), solidus (/), y signo igual (=), los caracteres de control , y los caracteres que no se definen por Unicode. Los atributos se colocan dentro de la etiqueta de inicio, y constan de un nombre y un valor , separados por un " =carácter". El valor del atributo puede permanecer sin comillas si no contiene caracteres de espacio o cualquiera de " ' ` = . De lo contrario, tiene que ser citado el uso de comillas simples o dobles. El valor, junto con el " =carácter", se puede omitir por completo si el valor es una cadena vacía.
